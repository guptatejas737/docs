## 1. Get Your API Key

First, you'll need an API key. Go to your Waves Platform dashboard and generate a new key.

<Info>
  **Important:** Keep your API key secret! Do not share it publicly or commit it to GitHub.
</Info>

## 2. Make Your First Call

We'll use a simple text-to-speech request as our first call.

### Endpoint

`POST https://api.smallest.ai/v1/tts`

### Headers

| Key             | Value                   | Required |
| --------------- | ----------------------- | -------- |
| `Authorization` | `Bearer <YOUR_API_KEY>` | Yes      |
| `Content-Type`  | `application/json`      | Yes      |

### Request

This example will generate audio for the text "Hello, world."

<Tabs>
  <Tab title="cURL">
    ```bash
    curl -X POST [https://api.smallest.ai/v1/tts](https://api.smallest.ai/v1/tts) \
    -H "Authorization: Bearer <YOUR_API_KEY>" \
    -H "Content-Type: application/json" \
    -d '{
          "text": "Hello, world."
        }'
    ```
  </Tab>
  <Tab title="Python">
    ```python
    import requests
    
    API_KEY = "<YOUR_API_KEY>"
    url = "[https://api.smallest.ai/v1/tts](https://api.smallest.ai/v1/tts)"
    
    headers = {
        "Authorization": f"Bearer {API_KEY}",
        "Content-Type": "application/json"
    }
    
    data = {
        "text": "Hello, world."
    }
    
    response = requests.post(url, headers=headers, json=data)
    
    if response.status_code == 200:
        # Handle successful audio response
        with open("hello_world.mp3", "wb") as f:
            f.write(response.content)
        print("Audio file saved!")
    else:
        print(f"Error: {response.status_code}")
        print(response.text)
    ```
  </Tab>
  <Tab title="JavaScript (Node.js)">
    ```javascript
    const fetch = require('node-fetch'); // or use native fetch
    
    const API_KEY = "<YOUR_API_KEY>";
    const url = "[https://api.smallest.ai/v1/tts](https://api.smallest.ai/v1/tts)";
    
    const headers = {
        "Authorization": `Bearer ${API_KEY}`,
        "Content-Type": "application/json"
    };
    
    const data = {
        "text": "Hello, world."
    };
    
    fetch(url, {
        method: 'POST',
        headers: headers,
        body: JSON.stringify(data)
    })
    .then(response => {
        if (response.ok) {
            return response.buffer(); // Handle audio stream
        }
        throw new Error(`HTTP error! status: ${response.status}`);
    })
    .then(buffer => {
        // Save buffer to a file
        console.log("Audio received!");
    })
    .catch(error => console.error('Error:', error));
    ```
  </Tab>
</Tabs>

### Success Response

If successful, the API will return the raw audio data. Your Python/JS code should handle saving this data as an `.mp3` or `.wav` file.

### Error Response

If your API key is invalid, you will receive a `401 Unauthorized` error.

```json
{
  "error": {
    "message": "Invalid API key.",
    "type": "authentication_error"
  }
}
```
